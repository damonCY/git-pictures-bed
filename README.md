# git-pictures-bed



利用github做图床

## Installation
```console
npm install git-pictures-bed -g
```

## Compatibility

pb -h
```js
Usage: pb [options]

Options:
  -V, --version      output the version number
  -p, --path <path>  上传指定路径下的文件
  --init <git>       初始化环境（git为git地址）
  --update           升级工具版本
  -h, --help         output usage information
```
